后端目录详情
com.quasar.art
├── config       // 配置信息 (AI API, Swagger等)
├── controller   // 控制层 (API 接口)
├── dto          // 数据传输对象 (输入输出模型)
├── entity       // 实体类 (数据库表映射)
├── repository   // 数据库操作 (JPA/MyBatis)
├── service      // 业务逻辑 (核心 AI/解析逻辑)
│   └── impl     // 业务实现
└── util         // 通用工具类 (PDF处理、字符串处理)
